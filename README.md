# 📌 Project Overview

Finding movies that match your interests can be overwhelming due to the vast number of titles available across streaming platforms.

This project uses a **Content-Based Recommendation System** to recommend movies with similar characteristics based on metadata such as genres, cast, crew, keywords, and plot descriptions. Users simply select a movie, and the system instantly recommends similar titles.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/shreya975/Movie-Recommendation-System.git
```

## Navigate into Project

```bash
cd Movie-Recommendation-System
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run the Streamlit Application

```bash
streamlit run app.py
```

The application will automatically open in your browser.

If it doesn't open automatically, visit:

```
http://localhost:8501
```
